# REDE-CIVIL

Projeto integrador Fapam 2° Periodo

# Sobre o Projeto

O projeto tem como objetivo o desenvolvimento de um protótipo de site de denúncias para a Defesa Civil – PM, buscando criar um canal direto de comunicação entre os cidadãos e os órgãos responsáveis. A proposta surgiu da necessidade de centralizar informações sobre problemas urbanos, como buracos em vias públicas, falhas de iluminação e outras ocorrências, permitindo o envio de descrições e imagens para tornar o processo mais ágil, acessível e transparente.

O sistema foi idealizado para facilitar o registro e acompanhamento das denúncias, reunindo as informações em uma única plataforma de forma organizada e intuitiva.

# Fluxo do Sistema

1. O usuário acessa o site e entra na área de denúncias.

2. A plataforma disponibiliza uma lista de ocorrências, separadas por níveis de risco, como denúncias emergenciais e progressivas, contendo descrições detalhadas sobre cada situação.

3. O usuário seleciona o tipo de denúncia desejado e acessa um formulário específico.

4. No formulário, o usuário deverá preencher:

   * Questionário relacionado à ocorrência;
   * Informações pessoais;
   * Descrição detalhada da denúncia;
   * Envio de imagens ou vídeos.

5. Após o envio, a denúncia é registrada no sistema e recebe um nível de urgência para priorização do atendimento.

6. O usuário poderá acompanhar o andamento da denúncia diretamente pela plataforma.

7. As denúncias serão revisadas por um agente responsável, que poderá analisar as informações e designar uma equipe para atendimento no local.
